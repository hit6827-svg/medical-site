# Medical Website Project

## Overview
This project is designed to provide users with comprehensive information about various medical conditions, treatments, and healthcare resources.

## Features
- **Condition Lookup**: Users can search for medical conditions and get detailed descriptions.
- **Treatment Information**: Insight into available treatment options for different conditions.
- **Healthcare Resources**: Links to reputable healthcare services and resources.

## Technologies Used
- HTML
- CSS
- JavaScript
- React
- Node.js

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hit6827-svg/medical-site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd medical-site
   ```
3. Install the required packages:
   ```bash
   npm install
   ```

## Usage
To start the application, run:
```bash
npm start
```
